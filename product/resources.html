More Color Resources

If you haven’t already, check out the crash course in color theory used for this project. Then, check out some more resources for color decision-making that can complement the color dice.

Other physical tools for understanding color theory include the Kolormondo, which can be assembled and disassembled using slot-fit cardboard. The Kolormondo represents a 3D color sphere version of color theory. You can learn more about the Kolormondo at their website.

Perception of color in design is largely based on cultural content. Learn more about color across cultures at this website.

Adobe's famous "Kuler" digital color wheel can be complicated to master, but you can also take color combinations from other users on their website. Check out Adobe Kuler here.

You may find a perfect fit from a collection of color palettes curated by a color expert. lolcolors is a great one.
